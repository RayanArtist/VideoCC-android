مرحله 3: انتقال و نصب
---------------------
# انتقال فایل به فولدر Download
cp VideoCC-Mobile-App.apk /sdcard/Download/

# یا مستقیماً نصب با adb (اختیاری)
# pkg install android-tools
# adb install VideoCC-Mobile-App.apk

مرحله 4: نصب دستی
-------------------
1. از Termux خارج شوید
2. به فولدر Download گوشی بروید
3. فایل VideoCC-Mobile-App.apk را پیدا کنید
4. روی فایل ضربه بزنید
5. "نصب از منابع ناشناخته" را فعال کنید
6. اپلیکیشن را نصب کنید

تکمیل!
-------
حالا Video.C.C روی گوشی شما نصب شده و آماده استفاده است.

مشخصات اپلیکیشن:
- نام: Video Communication Chat
- پکیج: com.videocc.app  
- نسخه: 1.0.0
- قابلیت‌ها: تماس ویدیویی، چت، VIP، توکن VCC، پخش زنده

نکات مهم:
----------
1. مطمئن شوید Termux از F-Droid نصب شده (نه Google Play)
2. دسترسی Storage را به Termux بدهید: termux-setup-storage
3. اگر خطای permission داشتید: chmod +x VideoCC-Mobile-App.apk
4. برای بررسی فایل: file VideoCC-Mobile-App.apk
